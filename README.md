# 🌌 NASA POD Stargazer

A modern astronomy web application that allows users to explore NASA's Astronomy Picture of the Day (APOD) archive. Users can search by date, discover random NASA images, save favorites, switch between dark and light themes, and learn about the science behind each featured image.

Built with vanilla JavaScript, HTML5, CSS3, and NASA's APOD API. NASA's APOD service provides daily astronomy images and explanations from professional astronomers and scientists.

## ✨ Features

### 📅 Search by Date

Select any available APOD date and instantly view NASA's featured image and explanation.

### 🌠 Random Stargazer Mode

Generate a random APOD date and discover astronomy images from NASA's archive.

### 🌞 Automatic Daily Loading

Today's Astronomy Picture of the Day loads automatically when the application starts.

### ⭐ Favorites System

Save favorite NASA images using Local Storage and revisit them later.

### 🔗 Copy Media Link

Quickly copy the current image URL to your clipboard.

### 📥 Download High Resolution Images

Open or download NASA's high-resolution image when available.

### 🌗 Theme Toggle

Switch between Dark Mode and Light Mode. Theme preferences persist across browser sessions.

### ⏳ Loading States

Visual loading indicators provide feedback while data is being retrieved.

### 📱 Responsive Design

Optimized for desktop, tablet, and mobile devices.

### 🛡 Error Handling

Gracefully handles invalid dates, unavailable media, and API errors.
