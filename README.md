# 🚀 NASA Astronomy Picture of the Day

A responsive web application that allows users to explore NASA's Astronomy Picture of the Day (APOD) by selecting a date and viewing the corresponding image, video, and description from NASA's public API.

## 🌌 Live Demo

🌐 Live Site:
https://lvcg.github.io/nasa-api-template-/

---

## 📸 Preview

The application allows users to:

- View NASA's Astronomy Picture of the Day
- Search APOD content by date
- Display images and videos dynamically
- Read detailed explanations provided by NASA
- Experience a responsive user interface optimized for desktop and mobile devices

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- NASA Open APIs

NASA's APOD endpoint provides daily astronomy images and metadata, including titles, explanations, dates, media types, and media URLs. :contentReference[oaicite:0]{index=0}
